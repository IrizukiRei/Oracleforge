# Audio Files for OracleForge

Place the following audio files in this folder:

## Required Files

| Filename | Description | Usage |
|----------|-------------|-------|
| `bgm.mp3` | Background music (loop) | Main screens, subtle ambience |
| `se_tap.mp3` | Button tap sound effect | Navigation, button clicks |
| `se_flip.mp3` | Card flip sound effect | Drawing cards, lottery |
| `se_flip_light.mp3` | Light card flip sound effect | 1-2 card flips |
| `se_flip_heavy.mp3` | Heavy card flip sound effect | 8+ card flips |
| `se_shuffle.mp3` | Shuffle sound effect | Shuffle action |
| `se_shuffle_animation.mp3` | Shuffle animation sound effect | Shuffle animation |
| `se_stack.mp3` | Card stacking sound effect | Collecting cards into a stack |
| `se_confirm.mp3` | Confirmation sound effect | Result display, spread complete |

## Recommended Specifications

- **Format**: MP3 (for browser compatibility)
- **BGM**: Loop-friendly, 30-120 seconds, 96-128kbps
- **SE**: Short (0.1-1 second), 128kbps

## Notes

- If files are missing, the app will work silently without errors
- Users can toggle sounds on/off in settings
- Volume can be adjusted (BGM: 0-100%, SE: 0-100%)
